**Scripting for Artists** is a series of introduction videos for artists into the different software packages like Maya, Nuke, 3ds Max and Python, MEL, MaxScript and TCL.
The main goal is to create an introduction intor scripting and keep you engaged all the way since basic scripting will enhance your working life and can erase a lot of frustation and repetition.
